# Aplica-o-de-controle-Fuzzy
Este trabalho tem como o objetivo apresentar uma aplicação de Controle Avançado de Processos para otimização do consumo de combustível. O objetivo específico é reduzir as variações de pressão de vapor, bem como atender a demanda de vapor necessária a produção.
